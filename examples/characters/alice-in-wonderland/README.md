# Alice in Wonderland — Tenniel Illustrations

Reference images of Alice from John Tenniel's original 1865 illustrations
for *Alice's Adventures in Wonderland* by Lewis Carroll.

All images are **public domain** (artist died 1914, 110+ years ago).

Source: [Wikimedia Commons](https://commons.wikimedia.org/wiki/Category:John_Tenniel%27s_illustrations_of_Alice%27s_Adventures_in_Wonderland)
